API Integration

Integrate the Codenix General Model chatbot into your website with ease. We offer two simple methods: a floating bubble or a direct embed.

Method 1: Floating Bubble Chatbot

Add a floating chat bubble to the corner of your site. Just copy and paste this snippet into your HTML file before the closing `</body>` tag.

HTML Snippet 
<script>
 window.difyChatbotConfig = {
  token: 'EeVGRIHe8CZyONST',
  inputs: {},
  systemVariables: {},
  userVariables: {},
 }
</script>
<script
 src="https://udify.app/embed.min.js"
 id="EeVGRIHe8CZyONST"
 defer>
</script>
<style>
  #dify-chatbot-bubble-button {
    background-color: #1C64F2 !important;
  }
  #dify-chatbot-bubble-window {
    width: 24rem !important;
    height: 40rem !important;
  }
</style>

Method 2: Embedded Window

Embed the chatbot directly into a section of your webpage using an `iframe`. This gives you more control over the layout.

iFrame Snippet 
<iframe
  src="https://udify.app/chatbot/EeVGRIHe8CZyONST"
  style="width: 100%; height: 600px; border: none;"
  allow="microphone"
>
</iframe>
